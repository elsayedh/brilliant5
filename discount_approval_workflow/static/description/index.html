Sale Order Discount Approval Workflow

Available Key Features
  • This module allows restrict discount on sale order line.
  • Discount Approval Manager Can allow user to give define discount only in sales order.
  • if User give more discount then allowed then Sales order goes for approval with Discount Approval Manager.
  • All Discount Approval Manager get approval email with order link and they can approve order.
  • Discount Approval Manager who approve the discount is linked with Approved Sales order.
1. Discount Approval Manager Group


Allow Discount in User, By setting limit of discount user can not confirm order if discount is more then this limit.


allow discount on sale order line from sale configuration.
2. Discount Approval Workflow: Give More discount then the allowed

A). User Creates Sale Order with discount


B). Once Discount is added more then allowed limit then try s to confirm order, sale order goes to approval stage and approval email goes to all users with Discount Approval Manager group.

3. Discount Approval Workflow: Approve Discount

A). Get an Email with Approval request to all user with Discount Approval Manager group


B). Discount Approval Manager can approve or cancel order.

C). Discount Approval Manager user linked with current approved order.
