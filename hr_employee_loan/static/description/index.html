Employee Loan Management in Odoo

This app allows your employee(s) to create loan requests in Odoo and allow the Loan Department (Loan Officer/ Loan Manager) to view all the loan requests and approve it and after approval accounting department can disburse loan to employee.


Main Features:

  • Allow employees to create Loan Requests and Loan Officer/Manager to Approve it.
  • Allow the accounting department to disburse Loan.
  • Allow you to print a PDF Loan Request Report.
  • Allow you to set up Loan Proofs.
  • Allow you to set up Loan Types Configuration.
  • Allow you to set up Loan Policy Configuration.
  • Repayment of Loan Installment by Cash/Bank method only available in this module.
  • Loan Disbursement using Cash/Bank.
  • Only company currency loan supported by app. So loans must be in company currency and installments are also in company currency only.
  • Loan Installments and Booking Interest and Booking entry if repayment method = cash/bank.
  • For more details please see below screenshots and watch the video.
  • Note: If you are using odoo enterprise edition or odoo.sh and if you want to have payroll as repayment method also on loan type then you can check app https://apps.odoo.com/apps/modules/14.0/employee_loan_payroll_enterprise/ and if you are ok with that you can buy it also separately.

Loan Proofs

This loan proof menu allow HR manager to create and configure all loan proofs that will be required to approve loan and process it. These loan proofs will be selected on loan types and then it will be visible on loan request form.

Loan Types List View

Loan Types Form View

Loan Types Form View

This loan types allow HR manager to create different types of loan for employees. Also this loan type allow user to configure loan given to employee will be interest payable or not.

Also it will allow to configure that loan type repayment method will be from only one direct cash/bank cheque from Employee at time of installment payment. This also allow to configure disburse method will be only by cash/bank. (Disbursement can be done when loan request will be approved.)

Here HR manager can list out Employee categories and employees which can request for this type of loan from company..... For example if Employee A is not eligible and not come under category selected in loan types then system will not allow him/her to submit loan request. Loan proof also configured here so that we can state that these proofs will be needed to process loan request for employee.

Loan Policies List View

HR manager can configure policies or rules for loan request and based on that rules it will check before approving loan that if employee full fill that policy or not.

Loan Policies Form View
